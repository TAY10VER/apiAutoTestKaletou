# 安装虚拟环境内依赖包
pycharm Terminal命令行输入：
>>pip install -r requirements.txt

(一) 项目目录结构
    1：api  	   初始化接口，定义请求url、请求数据
    2：cases	   测试脚本
    3：data        存放测试数据
	4：log         存放脚本运行日志
	5：report	   存放测试报告
	6：tools       存放工具类
	7：pytest.ini  pytest测试框架配置文件
	8：requirements.txt  项目依赖工具包

(二) 运行方式

    1，pycharm Terminal命令行输入
       >> pytest
